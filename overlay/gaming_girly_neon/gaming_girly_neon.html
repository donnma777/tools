// HOW TO USE
パーツの追加
ダブルクリック何もない場所でパーツ追加メニューを開く
パーツの操作
ヘッダーをドラッグボックスを移動
角・辺のハンドルホバーで表示、ドラッグでリサイズ
≡ ボタンボックスにホバーで右上に出現 → 複製・文字サイズ・削除
ヘッダー上部パーツホバーで右上に≡が出現 → 同様のメニュー
プロフィール欄
ダブルクリックプロフィールテキストを編集（複数行を自動で切り替え表示）
設定の保存・読込
設定を保存現在の配置をブラウザに保存（次回自動読込）
エクスポート設定をJSONファイルとしてダウンロード
インポート保存したJSONを読み込んで復元
表示更新同じURLを別タブで開いている場合、その画面へ即座に反映（同一オリジンで開く必要あり）
